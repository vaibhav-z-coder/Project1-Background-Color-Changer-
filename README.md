# 🎨 Color Changer Web App

A simple and interactive **Color Changer Web App** built using **HTML, CSS, and JavaScript**.  
This app allows users to dynamically change the **entire page background color** using random colors or custom input.

---

## 🚀 Features

- 🎲 Generate random colors from a predefined palette  
- 🎯 Apply custom colors (Hex or color name)  
- 🌈 Full-page background color change  
- ⚡ Instant UI update  
- 📱 Responsive and clean design  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
